SAFETY GUIDE

IMPORTANT WARNINGS

  • Inside the digital totems there is high voltage. Please do not open the cover to avoid electric shocks.
  • Maintenance of digital totems must be performed by qualified technical personnel.
  • Improper operation can lead to electric shocks or fires.

General Safety Instructions

Before using this unit, please read and understand the operating instructions first.

Store this instruction manual properly for future needs.

Pay special attention to safety indications and warnings and strictly comply with warning instructions in the manual.

Do not use accessories that the manufacturer does not recommend; improper use may lead to unexpected malfunction of the totem.

Use the totem with the supplied power cable. Check the power requirements if they are suitable for local conditions.

Do not use any object that presses or twists the power cable. A damaged cable can cause electric shock or fire.

Do not attempt to repair the totem yourself. Make sure you use qualified personnel for equipment maintenance.

Do not touch components that are not mentioned in the user manual.

Cleaning and Maintenance

Before cleaning the device:

  • Disconnect the power supply
  • Use a damp cloth to clean the device
  • Do not use liquid or aerosol cleaning products

When the unit will not be used for an extended period, please disconnect the power supply.

Make sure there is ventilation in the room and the ventilation slot of the device is not blocked.

CONNECTIONS

Wired Network Connection (Ethernet)

For a stable and fast connection, we recommend using an Ethernet cable connection.

Ethernet Connection Steps:

  1. Locate the Ethernet port on the back of the totem
  2. Connect the UTP/Ethernet cable to the port
  3. Connect the other end to the router/network switch
  4. The system will automatically detect the connection
  5. Check connection status in network settings

WARNING

Do not disconnect the network cable while the totem is operating and processing important data.

Wireless Network Connection (Wi-Fi)

For installations where Ethernet cable is not available, you can use Wi-Fi connection.

Wi-Fi Connection Steps:

  1. Access network settings from the operating system
  2. Select "Wi-Fi" or "Wireless Networks"
  3. Search for the desired network from the available list
  4. Enter the access password
  5. Wait for connection confirmation
  6. Test the connection by accessing the internet

Network Setup Tips

Security

  • Use networks with strong password
  • Avoid unsecured public networks
  • Update passwords regularly

Performance

  • Prefer wired connection
  • Check Wi-Fi signal strength
  • Ensure sufficient bandwidth

Configuration

  • Note the totem's IP address
  • Configure static IP if necessary
  • Test connectivity regularly

Cleaning the Totem

WARNING

Before cleaning, always disconnect the power supply!

Screen and Frame

1
Preparation

Turn off and disconnect the totem from power. Wait for it to cool completely.

2
Screen Cleaning

Use a soft microfiber cloth, slightly dampened with water. Do not spray directly on the screen.

3
Frame

Wipe the frame with a damp cloth. For persistent stains, use diluted neutral detergent.

4
Drying

Wipe with a dry cloth and let it dry completely before restarting.

Do not use:
  • Abrasive cleaning products
  • Aerosols or solvents
  • Rough materials that can scratch
  • Excess water that can penetrate the device

Housing and Support

  • Clean with a damp cloth
  • Periodically check mounting screws
  • Remove dust from ventilation areas

Power Cable

  • Visually inspect for damage
  • Do not bend or pull excessively
  • Replace immediately if damaged

Avoiding "Image Burn" or "Burn-in" Effect

To prevent screen damage from prolonged display of the same images:

  • Use screen saver after periods of inactivity
  • Vary the displayed content
  • Avoid displaying static images for long periods
  • Use screen saving settings from SITEKIOSK ONLINE

TROUBLESHOOTING

General

Totem cannot be turned on

Solutions:
  • Check if it is connected to outlet and outlet is working
  • Check the switch on the power cable
  • Make sure the power button is fully pressed
  • Contact technical service if problem persists

No image is displayed and no sound is produced

Solutions:
  • Check internal video connections
  • Make sure the screen is turned on
  • Check video input settings
  • Restart the system

Images appear slowly when totem starts

Explanation:

This is normal behavior at startup. The system needs time to load all components.

Cannot connect external devices

Solutions:
  • Check if device is compatible
  • Try another USB port
  • Restart totem with device connected
  • Check drivers in operating system

Totem turns off suddenly

Possible causes:
  • Overheating - check fans
  • Power problems - check connections
  • Power saving settings - check configuration
  • Hardware problems - contact service

Audio

No sound is produced while running

Solutions:
  • Check system volume
  • Make sure sound is not muted
  • Check speaker connections
  • Test with another audio file

No output from one of the speakers

Solutions:
  • Check audio balance (left/right)
  • Test with stereo content
  • Check speaker connections
  • Contact service for hardware check

Video

Horizontal or vertical bars appear or images fade

Solutions:
  • Check video cables
  • Adjust resolution and refresh rate
  • Remove electromagnetic interference sources
  • Contact service if problem persists

Ghosting images appear when product is turned off

Explanation:

It may be a normal temporary effect. If it persists, it could indicate burn-in. Use varied images and activate screen saver.

Power is on but screen appears extremely dark

Solutions:
  • Adjust screen brightness
  • Check power saving settings
  • Check if backlight is working
  • Contact service for diagnosis

"No Signal" appears on screen

Solutions:
  • Check connection between PC and screen
  • Make sure PC is turned on
  • Check selected input source
  • Reconnect video cables

Outdoor Equipment Installation

1

Foundation Preparation

For installing floor-mounted Outdoor equipment, it is necessary to prepare a perfectly flat concrete surface (if it will be mounted on a soft ground area).

Totem Digital Romania provides the dimensions for the concrete foundation, depending on the equipment.

2

Base Installation

Usually, the base is separate from the rest of the equipment, being the first part of the whole to be mounted on the foundation.

In the equipment box you will receive all screws necessary for fastening in the concrete foundation.

3

Cable Preparation

Make sure you have pulled through the center of the foundation a tube to provide:

  • 220V voltage cable
  • UTP cable for internet network
  • Condensate drain outlet (for air conditioning equipment)
4

Main Body Mounting

After installing the equipment base, follow:

  1. Positioning its body on the base
  2. Attaching to the base with screws provided in the package
  3. Fastening is done through the interior of the equipment
5

Using Lifting Ears

The equipment is provided with metal ears at the top, to be anchored using a lifting device, their weight being considerable.

After positioning and fixing on the base, the ears will be detached and kept for possible equipment relocation.
